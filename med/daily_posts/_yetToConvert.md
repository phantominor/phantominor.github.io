# 1 - 取卵手術（TVOR, Transvaginal Oocyte Retrieval）

1. 卵子數量與處理方式

- 若患者僅有一顆較成熟的濾泡並希望凍卵，則可選擇凍存此顆較大者，較小的濾泡則可透過 ^double stimulation^-(link to #3) 方式促進發育，待下次取卵。
- 卵子受精方式依活動力決定，常見方式：
- 體外受精（IVF）：適用於精子活動力佳的情況，讓精子自然受精。
- 顯微受精（ICSI, Intracytoplasmic Sperm Injection）：適用於精子活動力較低時，人工將精子直接注入卵母細胞內。

2. 取卵技術與處理

- 沖洗技術（Flushing）：
    - 可能使用濾泡同質 medium（如 Quinn’s Advantage Sequential Media, medium with HEPES）來沖洗濾泡，幫助回收卵母細胞。
    - 目前無明確證據顯示沖洗可提高受精率，但可能有助於取卵操作的順利進行。

3. ^顯微鏡下卵母細胞評估^-(link to #4)

- 卵母細胞成熟度可依細胞排列密度區分 (根據取卵當下技術員口述，後續資料搜尋整理後述)：
    - Grade A/B（較緊密）：通常較不成熟。
    - Grade C（較疏鬆）：通常較成熟。

4. 取卵手術技術

- 經^陰道後穹（Posterior fornix）^-(link to #5)進針，可能需要壓腹部以使卵巢下降，讓針頭更容易到達濾泡位置。

# 2 - 胚胎植入（ET, Embryo Transfer）

1. 胚胎植入時機

- 植入時機影響：
    - 若僅有 1 顆胚胎，通常儘早植入以提高存活率。
    - 若有多顆胚胎，可能培養較長時間，以挑選外觀最佳的進行植入。
    - 法規規範：不得植入超過 7 天的培養胚胎，以防止不當用途。

2. 胚胎植入技術

- 使用 UTM（Universal Transfer Medium）作為胚胎傳輸培養液。
- 輔助孵化（Assisted Hatching）：可能透過胚胎凝膠或其他方式促進胚胎著床。
- 植入位置：
    - 透過超音波於體表掃描，確保植入準確位置。
    - 胚胎通常放置於子宮頂下方 1～1.5 公分處。

3. 胚胎植入導管設計

- 使用 外管、內管 兩層結構：
    - 內管為軟管，上有四條刻度，每條刻度間隔 1 公分。
    - 內管第一條刻度，代表內管剛好與外管內口齊平，後續刻度則代表深入子宮的深度。
- 硬的外管進入後，會用一個塞子附件固定於 external os（子宮頸外口），可於超音波下顯影為亮白色，確保準確定位。

4. 胚胎植入確認

- 胚胎抽取技術：
    - 在培養液前後各留一小段空氣（總量僅約 0.04cc），確保胚胎被順利推入子宮。
    - 空氣間隔亦可幫助超音波顯影（空氣在超音波下為白色），以確認植入位置。

5. 胚胎篩選與基因檢測

- 若患者有高遺傳疾病風險或年齡較大，可能會考慮 ^PGT（Preimplantation Genetic Testing）^-(link to #6)，對胚胎進行基因切片檢測，以排除染色體異常或遺傳病變的胚胎。

# 3 - 何謂 Double Stimulation（雙重刺激排卵）？
Double Stimulation（DuoStim, 雙重刺激排卵） 是指在 同一個月經週期內進行兩次排卵刺激，即在濾泡期（follicular phase）及黃體期（luteal phase）各進行一次刺激，兩次都進行取卵。

傳統的促排卵（Controlled Ovarian Stimulation, COS） 通常只在濾泡期進行一次，而 Double Stimulation 則在第一次取卵後，立即開始第二次刺激，目標是在同一個週期內獲取更多卵子。

## 3.1 - Double Stimulation 的作用機制
- 以往認知：女性每個月經週期只有一波濾泡生長，選定的優勢濾泡發育，其他則凋亡（atresia）。
- 新發現： 研究顯示，同一個月經週期內可能有多波濾泡生長，因此在第一次取卵後，再次進行刺激，仍可誘導另一批濾泡發育。
- 這種策略適用於卵巢儲備功能較低（Diminished Ovarian Reserve, DOR）或急需累積足夠卵子進行胚胎培養的患者，如：
    - 高齡患者（>35-40 歲）
    - 卵巢早衰（POI, Premature Ovarian Insufficiency）
    - 低卵巢反應患者（如 AMH 低或基礎 AFC 低）
    - 癌症患者需趕在治療前凍卵

## 3.2 - Double Stimulation 的一般做法
1. 第一次刺激（follicular phase stimulation）
    - 在月經第2-3天開始使用GnRH拮抗劑或GnRH促效劑，搭配 FSH 或 hMG，促進濾泡發育。
    - 當濾泡成熟後，使用 hCG 或 Lupron 觸發排卵，並進行第一次取卵（TVOR）。
2. 第二次刺激（luteal phase stimulation）
    - 在第一次取卵後的1-5 天內，立即開始第二次排卵刺激。
    - 使用較高劑量的 FSH 或 hMG，因為黃體期內高量的孕酮可能會影響 FSH 敏感性。
    - 通常不會等待下一個月經週期，直接在黃體期進行刺激，約 10-12 天後取第二批卵。
3. 胚胎培養與冷凍
    - 由於黃體期內子宮內膜不適合立即植入，通常不會進行新鮮胚胎植入，而是先將胚胎冷凍（FET, Frozen Embryo Transfer），等後續週期內膜適合時再植入。

## 3.3 - Double Stimulation 是常規做法嗎？
不是所有人都需要 Double Stimulation，它通常適用於卵巢儲備低下或需要短時間內累積較多卵子的特殊患者。

對於**卵巢功能正常、能一次獲得足夠卵子**的患者，一般仍會使用單次促排卵，因為：
- 雙重刺激可能會增加荷爾蒙波動與副作用（如腫脹、不適）。
- 並非所有人對黃體期刺激都有良好反應，可能影響卵子品質。
- 對於**高齡或低反應者**，這種方法可大幅提高累積卵子的效率，是目前輔助生殖技術（ART）中一個重要的策略。

# 4 - 卵母細胞成熟度（Oocyte Maturity）分那些等級？
在取卵後，胚胎學家會檢查卵母細胞的成熟度，通常分為以下幾個等級：

+ table: [
成熟度	簡稱	特徵	能否受精    說明
成熟（MII）	Metaphase II	可見第一極體（Polar Body），細胞質正常	✅ 適合 ICSI 或 IVF
不成熟（MI）	Metaphase I	無極體，仍處於減數分裂中期	❌ 可能培養至 MII 再進行受精

在 IVF 或 ICSI 時，只有 **MII（成熟卵母細胞） 才能成功受精**，因此如果取卵時有很多 MI 或 GV，可能會考慮 ^Double Stimulation^-(link to #3) 或其他策略來增加成熟卵數量。

不過在 TVOR 手術期間，觀察技術員收集卵子的過程與紀錄，上面有一格是要記錄 grade A, B, C, D 四選一，而根據技術員口述，grade D (higher) 表示卵子越成熟，grade A 反之。而這種分級方式通常是藉由觀察卵子周圍的顆粒細胞（Cumulus Cells）緊密度而定，愈疏鬆表示愈成熟。

+ note: [
成熟的 MII 卵母細胞會有較為疏鬆的顆粒細胞（Cumulus Expansion），因為 FSH 促進透明帶周圍細胞的擴張。
較緊密的顆粒細胞可能代表該卵母細胞尚未成熟（MI 或 GV）。
這種視覺判讀有時會與實際成熟度稍有不同，因此仍需結合極體（Polar Body）觀察與 ICSI 受精結果來確認。

# 6 - 何謂 PGT？是拿胚胎去切片嗎？

PGT（Preimplantation Genetic Testing，植入前基因檢測） 是在 試管嬰兒（IVF） 過程中，對 囊胚（Day 5/6） 進行基因篩檢，以檢查胚胎是否存在染色體異常或遺傳疾病，確保選擇健康的胚胎進行移植，從而提高懷孕率並降低流產風險。

PGT 的分類
PGT 主要分為以下三種類型，每種用途不同：

+ table: [
類型	全名	檢測內容	適用對象
PGT-A	Preimplantation Genetic Testing for Aneuploidy	染色體異常（如 唐氏症、愛德華氏症、巴陶氏症）	高齡女性（>35 歲）、多次流產、不明不孕
PGT-M	Preimplantation Genetic Testing for Monogenic Disorders	單基因遺傳病（如 地中海貧血、囊腫性纖維化）	家族有已知遺傳病史
PGT-SR	Preimplantation Genetic Testing for Structural Rearrangements	染色體結構異常（如 羅氏易位、倒位）	父母帶有染色體結構異常
]

## PGT 的檢測流程
PGT 技術需要在胚胎發育至 囊胚期（Day 5/6） 進行 胚胎切片（Biopsy），取少量細胞進行基因分析，流程如下：

1. 體外受精（IVF/ICSI）
取出成熟卵母細胞，經由 IVF（體外受精）或 ICSI（單精子顯微注射） 讓卵子受精。
2. 胚胎培養至囊胚期（Day 5/6）
胚胎發育至 囊胚期（Blastocyst），此時已分化為：
- 內細胞群（ICM） → 發育成胎兒
- 滋養外胚層（TE） → 發育成胎盤
3. 胚胎切片（Biopsy）
取出 5-10 個 TE 細胞（不影響 ICM） 進行基因檢測。
由於 TE 細胞最終形成胎盤，因此對胚胎發育影響較小。
4. 冷凍保存（Vitrification）
胚胎切片後，胚胎會 冷凍保存，等待檢測結果。
5. 基因分析
- PGT-A（篩選染色體異常）：利用 NGS（次世代定序）或 aCGH（微陣列比較基因組雜交） 分析胚胎是否為 染色體異常（Aneuploidy）。
- PGT-M（篩選單基因疾病）：根據 家族遺傳基因突變 設計專屬檢測，篩選健康胚胎。
- PGT-SR（篩選結構異常）：分析 染色體異常排列（如倒位、易位），確保選擇正常胚胎。
6. 選擇健康胚胎移植
只移植染色體正常（Euploid）的胚胎，提升成功率。
- PGT 的優點
    - 提高懷孕率：選擇 染色體正常胚胎，提高植入率與活產率。
    - 降低流產風險：減少因 染色體異常 導致的 胚胎停育 或 早期流產。
    - 避免遺傳病：家族有單基因遺傳病時，可透過 PGT-M 確保胚胎健康。
    - 適用於高齡女性：35 歲以上婦女，卵子染色體異常風險上升，PGT 可篩選 正常胚胎，提高成功率。

- PGT 的缺點與爭議
    - ⚠ 費用高昂：PGT-A 一次約 10-15 萬台幣，PGT-M 可能更貴。
    - ⚠ 可能影響胚胎：胚胎切片（Biopsy）可能影響發育，但現代技術風險已降低。
    - ⚠ 非 100% 準確：嵌合體（Mosaicism） 可能導致檢測錯誤，部分胚胎可能仍然健康。
    - ⚠ 倫理與法律爭議：^台灣 PGT-A^-(link to #7) 需經醫學遺傳諮詢，^無法隨意篩選性別^-(link to the following note part)。

- PGT 適合哪些人？
    - 35 歲以上的高齡婦女（卵子染色體異常風險高）
    - 多次流產或反覆植入失敗（RIF）
    - 不明原因的不孕症
    - 家族有遺傳病史（如地中海貧血）
    - 有染色體結構異常（如羅氏易位）

+ note: [
除非與遺傳疾病相關（如 X-linked hereditary disease，可透過選擇男孩來篩除來自母親的帶原基因），台灣不允許透過 PGT 選擇胚胎性別，僅能篩選健康胚胎。
]
